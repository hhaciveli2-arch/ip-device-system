# IP Device Management System

This project was developed during an internship.

It is a system designed to manage and monitor IP-based devices.

## Features
- Device tracking
- Backend system with Flask
- Authentication system
- Inventory management

## Technologies
- Python (Flask)
- SQLAlchemy
- HTML / CSS

## Setup

1. Install dependencies:
pip install -r requirements.txt

2. Create environment file:
copy env.template.txt to .env

3. Run the project:
python backend/app.py

## Notes
All data in this project is mock data for demonstration purposes.
